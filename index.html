ChatBot Telegram

A Telegram chatbot is a computer program that can automatically reply to messages on Telegram. Telegram bots work 24/7 and can have multiple conversations with several people, at the same time.

WordPress

WordPress is a free and open-source content management system written in hypertext preprocessor language and paired with a MySQL or MariaDB database with supported HTTPS.

React

React is a free and open-source front-end JavaScript library for building user interfaces based on components. It is maintained by Meta and a community of individual developers and companies.
